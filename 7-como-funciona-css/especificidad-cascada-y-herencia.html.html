Especificidad, cascada y herencia

Especificidad

Establece lo especifico de un elemento para saber que estilo aplicar.

Formula:

001 => Etiquetas y pseudoelementos

010 => clases, atributos y pseudoclases

100 => Ids

1000 => Estilos en linea

***"!important" tiene priodad en todos, pero no se debe usar nunca.

Cascada

Lee el contenido de arriba hacia abajo para aplicar el estilo.

Funciona siempre que la especificidad tenga el mismo valor en los elementos.

Herencia

Es la capacidad de un elemento de heredar sus propiedades, con "inherit" se obtiene explicitamente la herencia, y con initial se obtiene las propiedades por defecto.
