Especificidad, cascada y herencia

Inicio

Especificidad

  • Establece lo especifico de un elemento para saber que estilo aplicar.

Formula:

  • 0001 => Etiquetas y pseudoelementos
  • 0010 => clases, atributos y pseudoclases
  • 0100 => Ids
  • 1000 => Estilos en linea
                                
                                    Codigo:

                                    (valor de especificidad = 0001)
                                    h1{
                                        background-color: rgb(77, 115, 185);
                                    }

                                    (valor de especificidad = 0111) <=== este es el que tiene mayor prioridad
                                    h1.title#title{
                                        background-color: #3a6ba3;
                                    }
                                    
                                    (valor de especificidad = 0011)
                                    h1.title{
                                        background-color: cornsilk;
                                    }
***"!important" tiene priodad en todos, pero no se debe usar nunca.

Cascada

  • Lee el contenido de arriba hacia abajo para aplicar el estilo.
  • Funciona siempre que la especificidad tenga el mismo valor en los elementos.
                            
                                Codigo:
                                .h1{
                                    font-size: 0.5rem;
                                }
                                .h1{
                                    font-size: 1rem;
                                }
                                .h1{
                                    font-size: 1.5rem;<=== este es el que tiene mayor prioridad
                                }

Herencia

  • Es la capacidad de un elemento de heredar sus propiedades, con "inherit" se obtiene explicitamente la herencia, y con initial se obtiene las propiedades por defecto.
                            
                                Codigo:
                                p{
                                    color: #2b999c;
                                }
                                p span{
                                    color: inherit;<== hereda el color del padre.
                                }
