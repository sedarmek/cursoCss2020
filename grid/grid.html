Parte 1: Medidas de los items

  • grid: engloba las propiedades grid,pero no se suele usar ya que no tiene soporte total.
  • grid-template: shorthand que engloba grid-template-columns,grid-template-rows y grid-template-areas
    Sintaxis:
    grid-template: [grid-areas | grid-template-rows] / grid-template-columns
  • grid-template-areas: define el nombre de los items,el numero de columnas(contando los items dentro de las comillas) y el numero de filas(contando las cadenas de texto con comillas) usando una cadena de texto dentro de comillas dobles.
    Sintaxis:
    grid-template-areas: "nombre"
  • grid-template-columns: forma explicita que define el nombre de las lineas, el tamaño y numero de columnas.
    Sintaxis:
    grid-template-columns: [line-name-column] valor
  • grid-template-rows: forma explicita que define el nombre de las lineas, el tamaño y numero de filas.
    Sintaxis:
    grid-template-rows: [line-name-row] valor
  • grid-auto-columns: forma implicita que define el tamaño de las columnas.
    Sintaxis:
    grid-auto-columns: valor
  • grid-auto-rows: forma implicita que define el tamaño de las filas.
    Sintaxis:
    grid-auto-rows: valor
    • Valores:
      • fr: indica las divisiones que recibira cada elemento hijo
      • auto: genera el tamanio minimo de acuerdo al contenido del elemento hijo
      • min-content: minimo necesario en funcion del contenido
      • max-content: maximo necesario en funcion del contenido
      • auto-fill: crea tantos grid-items vacios quepan en el viewport respetando la medida.
      • auto-fit: elimina los grid-items vacios que no se esten ocupando
      • repeat(x, y z): repite un patron de medidas, donde X es el numero de repeticiones, e Y Z son medidas del patron.
      • minmax(min,max): recibe dos parametros, el minimo y maximo que pueden tener los items
      • fit-content(): ajusta un tamaño dado a un tamaño disponible de acuerdo a la fórmula: min(maximum size, max(minimum size, argument)).

Parte 2: Medidas de separaciones y Nombre de lineas

  • grid-gap: shorthand de las propiedades grid-column-gap y grid-row-gap.
    Sintaxis:
    grid-gap: grid-column-gap / grid-row-gap
  • grid-column-gap: define la separacion entre las columnas.
    Sintaxis:
    grid-column-gap: longitud
  • grid-row-gap: define la separacion entre las filas.
    Sintaxis:
    grid-row-gap: longitud
  • grid-column: shorthand de la propiedades grid-column-start y grid-column-end.
    Sintaxis:
    grid-column: grid-column-start / grid-column-end
  • grid-column-start: especifica la posición inicial de un elemento tomando como referencia la line-column.
    Sintaxis:
    grid-column-start: line-column
  • grid-column-end: especifica la posición final de un elemento tomando como referencia la line-column.
    Sintaxis:
    grid-column-end: line-column
  • grid-row: shorthand de las propiedades grid-row-start y grid-row-end.
    Sintaxis:
    grid-row: grid-row-start / grid-row-end
  • grid-row-start: especifica la posición inicial de un elemento tomando como referencia la line-row.
    Sintaxis:
    grid-row-start: line-row
  • grid-row-end: especifica la posición final de un elemento tomando como referencia la line-row.
    Sintaxis:
    grid-row-end: line-row

Parte 3: Direccion de flujo y alineamiento

  • grid-auto-flow: establece la direccion en la cual se posicionan los items.
    • row: se crearan filas adicionales.
    • column: se crearan columnas adicionales.
    • dense: establece que hacer con los huecos que queden.
    Sintaxis:
    grid-auto-flow: row | column | dense
  • justify-items: alinea los elementos horizontalmente respecto a la celda.
    Sintaxis:
    justify-items: valor
  • align-items: alinea los elementos verticalmente respecto a la celda.
    Sintaxis:
    align-items: valor
  • justify-content: alinea los elementos horizontalmente respecto al contenedor.
    Sintaxis:
    justify-content: valor
  • align-content: alinea los elemetos verticalmente respecto al contenedor.
    Sintaxis:
    align-content: valor
  • justify-self: alinea los elementos horizontalmente respecto a la celda, esta propiedad se establece en el item especifico.
    Sintaxis:
    justify-self: valor
  • align-self: alinea los elementos verticalmente respecto a la celda, esta propiedad se establece en el item especifico.
    Sintaxis:
    align-self: valor
  • Valores basicos:
    • left: alineacion a la izquierda
    • right: alineacion a la derecha
    • center: alinacion al centro
  • order: establece el orden de alineamiento, independientemente del orden establecido en el documento html.
    Sintaxis:
    order: numero
