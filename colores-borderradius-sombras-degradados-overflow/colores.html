Overflow

  • Especifica como se comporta el elemento respecto a la visibilidad de su contenido:
    • visible: Valor por defecto. El contenido no es recortado, podría ser dibujado fuera de la caja contenedora.
    • hidden: El contenido es recortado y no se muestran barras de posición.
    • scroll: El contenido es recortado y el navegador web usa las barras de desplazamiento, se haya recortado contenido o no.
    • auto: Depende del agente de usuario. Navegadores de escritorio como Firefox proveen barras de desplazamiento si hay contenido excedente.
Sintaxis:
overflow: valor
Overflow: auto
Overflow: scroll
                            
                                Codigo:
                                .item-overf-auto{
                                    background-color: #6060e2;
                                    overflow: auto;
                                }
                                .item-overf-scroll{
                                    background-color: #6060e2;
                                    overflow: scroll;
                                }

Border-radius

  • Redondea las esquinas del borde del borde exterior de un elemento. Puede establecer un solo radio para hacer esquinas circulares o dos radios para hacer esquinas elípticas.
  • Se especifica como:
    • Circular: uno, dos, tres o cuatro valores. Se utiliza para establecer un solo radio para las esquinas.
    • Eliptico: seguido opcionalmente por "/" y uno, dos, tres, o cuatro valores. Esto se usa para establecer un radio adicional, por lo que puede tener esquinas elípticas.
Sintaxis:
border-radius: longitud(1-4) / longitud(1-4)
Border-radius-circle
Border-radius-ellipse
                            
                                Codigo:
                                .item-border-circl{
                                    background-color: #54386d;
                                    border-radius: 1em 2em 3em 4em;
                                }
                                .item-border-ellip{
                                    background-color: #54386d;
                                    border-radius: 1em 2em 3em 4em / 3em;
                                }

Colores

  • Es un tipo de datos que denota un color.Puede ser descrito de cualquiera de las siguiente maneras:
    • Usando una palabra clave.
      • Son identificadores que sirven para representar un color concreto, donde el nombre describe el color.
        • Sintaxis:
        Palabraclave
    • Usando la notacion hexadecimal.
      • Los colores son definidos usando el modelo RGB, red-green-blue.
      • La notacion es: "#", seguido de seis o tres caracteres que pueden ser del 0 al 9 y de la A a la F.
        • Sintaxis:
        #valor
    • Usando la notacion rgb.
      • Los colores son definidos usando el modelo RGB(Red,Green,Blue).
      • Los valores pueden ser enteros del 0 al 255 o en porcentaje del 0 al 100%.
        • Sintaxis:
        rgb(valor,valor,valor)
    • Usando la notacion rgba.
      • Es similar que la notacion rgb, pero añade un canal nuevo para la opacidad, siendo el valor 0 totalmente transparente.
        • Sintaxis:
        rgb(valor,valor,valor,valor)
    • Usando el sistema hsl.
      • Los colores son definidos usando el modelo HSL(Hue,Saturation,Lightness)
      • El primer valor es un numero entero que representan la posicion en angulo de una paleta de colores ya definida, el segundo valor es un porcentaje que representa la saturacion, y el tercer valor es un porcentaje que representa el brillo.
        • Sintaxis:
        hsl(valor,valor,valor)
    • Usando el sistema hsla.
      • Es similar que la notacion hsl, pero añade un canal nuevo para la opacidad, siendo el valor 0 totalmente transparente.
        • Sintaxis:
        hsla(valor,valor,valor,valor)

Box-shadow y Text-shadow

  • El box-shadow añade efectos de sombra alrededor del marco de un elemento.
    Sintaxis:
    [offset-x | offset-y | blur-radius | spread-radius | color]
  • El text-shadow añade efectos de sombra alrededor del texto.
    Sintaxis:
    [offset-x | offset-y | blur-radius | color]
Box-shadow
Text-shadow

                                }

Linear gradient y radial gradient y repeating

  • Linear-gradient crea una imagen que consiste en una transición progresiva entre dos o más colores a lo largo de una línea recta.
    Sintaxis:
    angulo | colorInicial[positionInicial || positionFinal] | colorFinal[positionInicial || positionFinal]
  • Radial-gradient crea una imagen que consiste en una transición progresiva entre dos o más colores que irradian desde un origen. Su forma puede ser un círculo o una elipse.
    Sintaxis:
    circle at center || colorInicial[positionInicial | positionFinal] || colorFinal[positionInicial | positionFinal]
  • Reapeting-linear-gradient crea una imagen que consta de gradientes lineales repetidos. Es similar a linear gradient y toma los mismos argumentos, pero repite las paradas de color infinitamente en todas las direcciones para cubrir todo su contenedor.
    Sintaxis:
    angulo | colorInicial[positionInicial || positionFinal] | colorFinal[positionInicial || positionFinal]
  • Reapeting-radial-gradient crea una imagen que consiste en degradados repetidos que irradian desde un origen. Es similar a radial-gradient y toma los mismos argumentos, pero repite las paradas de color infinitamente en todas las direcciones para cubrir todo su contenedor.
    Sintaxis:
    circle at center || colorInicial[positionInicial | positionFinal] || colorFinal[positionInicial | positionFinal]
