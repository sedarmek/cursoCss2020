Box model

Inicio

Elemento de linea

  • Una etiqueta de línea es aquella que ocupa el espacio mínimo necesario en horizontal, y permite que otro elemento se coloque a su lado.

Etiquetas mas usadas: <a>, <span>, <strong>, <img>, <input>, <code>

Elemento de bloque

  • Una etiqueta de bloque, ocupa todo el ancho disponible y no permite que otro elemento se coloque a su lado.
  • Con este elemento es que funciona el box model.

Etiquetas mas usadas: <h1>, <p>, <ul>, <li>, <div>, <header>, <nav>, <section>, <article>, <footer>, <form>, <table>

Display

  • Cambia el contexto de los elementos dentro del navegador.
  • Admite varios valores:
    • none: hace que el elemento no se muestre, pero sigue cargandose.
    • block: hace que el elemento sea de bloque.
    • inline: hace que el elemento sea de linea.
    • inline-block: hace que el elemento sea de linea, pero admite medidas y margenes verticales.

Width y height

  • Son pripiedades que se usan para dar una medida de ancho y alto respectivamente.
  • Funcionan en los elementos de bloque, pero en los elementos de linea no.

Elemento con medidas

                                
                                    Codigo:
                                    .element-size{
                                        width: 7rem;
                                        height: 7rem;
                                    }

Margin

  • Es una propiedad que establece el margen para los cuatro lados.
  • Se puede establecer cada lado por separado con otras propiedades: margin-top, margin-right, margin-bottom y margin-left.
  • Acepta uno, dos , tres o cuatro valores de los siguientes:
    • Un único valor aplicará para todos los cuatro lados.
    • Dos valores aplicarán: El primer valor para arriba y abajo, el segundo valor para izquierda y derecha.
    • Tres valores aplicarán: El primero para arriba, el segundo para izquierda y derecha, el tercero para abajo.
    • Cuatro valores aplicarán en sentido de las manecillas del reloj empezando desde arriba.
  • Funciona en los elementos de bloque y en los elementos de linea, pero en estos ultimos solo con left y right.

Elemento con margen

                                
                                    Codigo:
                                    .element-margin-1{
                                        width: 5em;
                                        height: 5em;
                                        margin: 0.3em auto;
                                    }
*Para que un elemento se mueva a los lados al usar margin, debes asignarle un ancho. *El margin por defectos es de 8 pixeles.

Padding

  • Establece el espacio (entre el contenido del elemento y su borde) de relleno requerido por todos los lados de un elemento.
  • Se puede establecer cada lado por separado con otras propiedades: padding-top,padding-right,padding-left,padding-bottom.
  • Acepta uno, dos , tres o cuatro valores de los siguientes:
    • Un único valor aplicará para todos los cuatro lados.
    • Dos valores aplicarán: El primer valor para arriba y abajo, el segundo valor para izquierda y derecha.
    • Tres valores aplicarán: El primero para arriba, el segundo para izquierda y derecha, el tercero para abajo.
    • Cuatro valores aplicarán en sentido de las manecillas del reloj empezando desde arriba.
  • Funciona en los elementos de bloque y en los elementos de linea.

Elemento con padding

                                
                                    Codigo:
                                    .element-padding-1{
                                        width: 5em;
                                        height: 5em;
                                        margin: 0.3em auto;
                                        padding: 1em;
                                    }
*El padding puede modificar el tamaño del elemento.

Border

  • Es el borde el elemento, ye puede establecer cada lado por separado con otras propiedades: border-top,border-right,border-left,border-bottom.
  • Es una propiedad abreviada para las siguientes propiedades CSS:
    • border-width
    • border-style
    • border-color
  • Funciona en los elementos de bloque y en los elementos de linea.

Elemento con border

                                
                                    Codigo:
                                    .element-border-1{
                                        width: 5em;
                                        height: 5em;
                                        margin: 0.3em auto;
                                        padding: 1em;
                                        border: 1px solid black;
                                    }

Outline

  • Tiene similitudes con border, pero no esta dentro del modelo de cajas.
  • Diferencias: no tiene la opcion de trabajar cada lado por separado, no modifica el flujo de trabajo, puede ser interno o externo, y ademas, tiene un separamiento.

Elemento outline

                                
                                    Codigo:
                                    .element-border-1{
                                        width: 5em;
                                        height: 5em;
                                        margin: 0.3em auto;
                                        padding: 1em;
                                        outline: 1px solid white;
                                    }

Box-sizing

  • La propiedad debe calcular el ancho y alto total de un elemento.
  • Tiene dos valores:
    • content-box: valor por defecto, el padding puede modificar su medida.
    • border-box: calcula las dimensiones del contenido automaticamente cuando se modifica el padding o el border para que el elemento no cambie su tamaño.
  • Funciona en los elementos de bloque.

Elemento con box sizing

                                
                                    Codigo:
                                    .element-bsizing-1{
                                        width: 5em;
                                        height: 5em;
                                        margin: 0.3em auto;
                                        padding: 1em;
                                        border: 1px solid black;
                                        box-sizing: border-box;
                                    }

Border radius

  • Redondea las esquinas del borde exterior de un elemento. Puede establecer un solo radio para hacer esquinas circulares o dos radios para hacer esquinas elípticas.
  • Se puede establecer cada esquina por separado con otras propiedades: border-top-left-radius, border-top-right-radius, border-bottom-right-radius, border-bottom-left-radius.
  • Acepta uno, dos , tres o cuatro valores de los siguientes:
    • Un único valor aplicará para todas las esquinas.
    • Dos valores aplicarán: El primer valor para esquina superior izquierda/esquina inferior derecha, el segundo valor para esquina superior derecha /esquina inferior izquierda.
    • Tres valores aplicarán: El primero para esquina superior izquierda, el segundo para esquina superior derecha /esquina inferior izquierda, el tercero para esquina inferior derecha.
    • Cuatro valores aplicarán en sentido de las manecillas del reloj empezando desde esquina superior izquierda.
  • Funciona en los elementos de bloque.

Elemento border radius

                                
                                    Codigo:
                                    .element-b-radius-1{
                                        width: 5em;
                                        height: 5em;
                                        margin: 0.3em auto;
                                        padding: 1em;
                                        border-radius: 0.5em 1em 1.5em 2em / 0.3em;
                                    }
*El slash divide el radio en el eje x del eje y, para hacer radios elipticos.

Overflow

  • Especifica si recortar contenido, dibujar barras de desplazamiento o mostrar el contenido excedente en un elemento a nivel de bloque.
  • Valores:
    • Visible: valor por defecto. El contenido no es recortado, podría ser dibujado fuera de la caja contenedora.
    • Hidden: el contenido es recortado y no se muestran barras de posición.
    • Scroll: el contenido es recortado y el navegador web usa las barras de desplazamiento, se haya recortado contenido o no.
    • Auto: el contenido sera recortado y usa barras de desplazamiento solo si es necesario.
  • Funciona en los elementos de bloque.

Text align

  • Establece la alineación horizontal del contenido dentro de un elemento de bloque o cuadro de celda de tabla.
  • Valores:
    • start: Lo mismo que leftsi la dirección fuera de izquierda a derecha y rightsi la dirección fuera de derecha a izquierda.
    • end: Lo mismo que rightsi la dirección fuera de izquierda a derecha y leftsi la dirección fuera de derecha a izquierda.
    • left: El contenido en línea se alinea con el borde izquierdo del cuadro de línea.
    • right: El contenido en línea se alinea con el borde derecho del cuadro de línea.
    • center: Los contenidos en línea están centrados dentro del cuadro de línea.
    • justify: Los contenidos en línea están justificados. El texto debe estar espaciado para alinear sus bordes izquierdo y derecho con los bordes izquierdo y derecho del cuadro de línea, excepto por la última línea.
    • justify-all: Igual que justify, pero también obliga a justificar la última línea.
    • match-parent: similar a inherit, pero los valores start y end se calculan de acuerdo con los del elemento padre y se reemplazan por el valor apropiado left o right.
  • Funciona en los elementos de linea.

                                    .element-t-align-1 p{
                                        text-align: left;
                                    }

Box shadow

  • Añade efectos de sombra alrededor del marco de un elemento.
  • Acepta de 2 a 6 valores de los siguientes:
    • Si solo dos valores son definidos, son interpretados como valores "offset-x" y "offset-y"(valores para definir el desplazamiento de la sombra).
    • Si un tercer valor es dado, es interpretado como un "blur-radius"(difuminacion).
    • Si un cuarto valor es dado, es interpretado como un "spread-radius"(escalado positivo o negativo).
    • Opcionalmente, la palabra clave inset(se dibuja dentro del borde del elemento).
    • Opcionalmente, el valor "color".

Elemento box shadow

                                
                                    Codigo:
                                    .element-b-radius-1{
                                        width: 5em;
                                        height: 5em;
                                        margin: 0.3em auto;
                                        padding: 1em;
                                        border-radius: 0.5em 1em 1.5em 2em / 0.3em;
                                        box-shadow: 0.3em 0.2em 1em 0.5em inset grey;
                                    }
