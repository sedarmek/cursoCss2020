Box model

Inicio

Elemento de linea

  • Una etiqueta de línea es aquella que ocupa el espacio mínimo necesario en horizontal, y permite que otro elemento se coloque a su lado.

Etiquetas mas usadas: <a>, <span>, <strong>, <img>, <input>, <code>

Elemento de bloque

  • Una etiqueta de bloque, ocupa todo el ancho disponible y no permite que otro elemento se coloque a su lado.
  • Con este elemento es que funciona el box model.

Etiquetas mas usadas: <h1>, <p>, <ul>, <li>, <div>, <header>, <nav>, <section>, <article>, <footer>, <form>, <table>

Width y height

  • Son pripiedades que se usan para dar una medida de ancho y alto respectivamente.
  • Funcionan en los elementos de bloque, pero en los elementos de linea no.

Elemento con medidas

                                
                                    Codigo:
                                    .element-size{
                                        width: 7rem;
                                        height: 7rem;
                                    }

Margin

  • Es una propiedad que establece el margen para los cuatro lados.
  • Se puede establecer cada lado por separado con otras propiedades: margin-top, margin-right, margin-bottom y margin-left.
  • Acepta uno, dos , tres o cuatro valores de los siguientes:
    • Un único valor aplicará para todos los cuatro lados.
    • Dos valores aplicarán: El primer valor para arriba y abajo, el segundo valor para izquierda y derecha.
    • Tres valores aplicarán: El primero para arriba, el segundo para izquierda y derecha, el tercero para abajo.
    • Cuatro valores aplicarán en sentido de las manecillas del reloj empezando desde arriba.
  • Funciona en los elementos de bloque y en los elementos de linea, pero en estos ultimos solo con left y right.

Elemento con margen

                                
                                    Codigo:
                                    .element-margin-1{
                                        width: 5em;
                                        height: 5em;
                                        margin: 0.3em auto;
                                    }
*Para que un elemento se mueva a los lados al usar margin, debes asignarle un ancho. *El margin por defectos es de 8 pixeles.

Padding

  • Establece el espacio (entre el contenido del elemento y su borde) de relleno requerido por todos los lados de un elemento.
  • Se puede establecer cada lado por separado con otras propiedades: padding-top,padding-right,padding-left,padding-bottom.
  • Acepta uno, dos , tres o cuatro valores de los siguientes:
    • Un único valor aplicará para todos los cuatro lados.
    • Dos valores aplicarán: El primer valor para arriba y abajo, el segundo valor para izquierda y derecha.
    • Tres valores aplicarán: El primero para arriba, el segundo para izquierda y derecha, el tercero para abajo.
    • Cuatro valores aplicarán en sentido de las manecillas del reloj empezando desde arriba.
  • Funciona en los elementos de bloque y en los elementos de linea.

Elemento con padding

                                
                                    Codigo:
                                    .element-padding-1{
                                        width: 5em;
                                        height: 5em;
                                        margin: 0.3em auto;
                                        padding: 1em;
                                    }
*El padding puede modificar el tamaño del elemento.

Border

  • Es el borde el elemento, ye puede establecer cada lado por separado con otras propiedades: border-top,border-right,border-left,border-bottom.
  • Es una propiedad abreviada para las siguientes propiedades CSS:
    • border-width
    • border-style
    • border-color
  • Funciona en los elementos de bloque y en los elementos de linea.

Elemento con border

                                
                                    Codigo:
                                    .element-border-1{
                                        width: 5em;
                                        height: 5em;
                                        margin: 0.3em auto;
                                        padding: 1em;
                                        border: 1px solid black;
                                    }

Box-sizing

  • La propiedad debe calcular el ancho y alto total de un elemento.
  • Tiene dos valores:
    • content-box: valor por defecto, el padding puede modificar su medida.
    • border-box: calcula las dimensiones del contenido automaticamente cuando se modifica el padding o el border para que el elemento no cambie su tamaño.
  • Funciona en los elementos de bloque.

Elemento con box sizing

                                
                                    Codigo:
                                    .element-bsizing-1{
                                        width: 5em;
                                        height: 5em;
                                        margin: 0.3em auto;
                                        padding: 1em;
                                        border: 1px solid black;
                                        box-sizing: border-box;
                                    }
