Variables Css(Custom properties)

Inicio
  • Necesitan estar dentro de un selector.
  • Tienen herencia y cascada.
  • Pueden ser globales y locales.
                        
                            Sintaxis:
                            selector{
                                --variable:valor;
                            }
                            propiedad: var(--variable);
Elemento con variables locales
Elemento con variables globales
                            
                                Codigo:
                                :root{
                                    --backg-color-childrens: rebeccapurple ;
                                    --color-text: white;
                                }
                                .element-local{
                                    --backg-color-childrens: pink;
                                    --color-text: black;
                                    background-color: var(--backg-color-childrens);
                                    color: var(--color-text);
                                }
                                .element-global{
                                    background-color: var(--backg-color-childrens);
                                    color: var(--color-text);
                                }
