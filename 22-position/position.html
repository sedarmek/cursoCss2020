Position

Inicio
  • La propiedad Position nos permite posicionar los elementos.

Conceptos y propiedades basicas antes de usar position

Conceptos

Flujo de renderizado:
Para norma general los elementos se dibujan de izquierda a derecha y de arriba a abajo, por lo que el punto 0,0 es la esquina superior izquierda.
Espacio reservado:
Es el espacio que tiene un elemento en el navegador.
Elemento posicionado:
Elemento que tiene la propiedad position con un valor distinto de "static"(valor por defecto).
Stacking context:
El contexto de apilamiento es el orden de superposicion de los elementos.

Propiedades para mover los elementos

Top:
El elemento se movera desde la parte superior la distancia indicada.
Right:
El elemento se movera desde la parte derecha la distancia indicada.
Bottom:
El elemento se movera desde la parte inferior la distancia indicada.
Left:
El elemento se movera desde la parte izquierda la distancia indicada.
Z-index:
El elemento se movera en el contexto de apilamiento.
Nota: si un elemento ya tiene declarada la propiedad top la propiedad bottom no funcionaran.Y lo mismo sucede con left y right respectivamente.

Z-index

  • Indice que sirve para posicionar un elemento sobre otro o visceversa.
  • Puede tener numeros enteros negativos y positivos

Stacking context

  • 1. Elemento posicionado con z-index
  • 2. Elemento posicionado sin z-index
  • 3. Elemento no posicionado
  • 4. Elemento con z-index negativo

Valores de position

Static

  • Valor por defecto.
  • No se puede mover el elemento, ya que no esta posicionado.
Elemento estatico
                                
                                    Codigo:
                                    .element-static{
                                        position: static;
                                    }

Relative

  • Conserva el espacio reservado.
  • Se coloca respecto a su misma posicion.
Espacio reservado
Elemento relativo
                                
                                    Codigo:
                                    .element-relative{
                                        position: relative;
                                        top: 25%;
                                        left: 40%;
                                    }

Absolute

  • No conserva el espacio reservado.
  • Se coloca respecto al contenedor posicionado mas cercano.
Elemento absoluto
                                
                                    Codigo:
                                    .element-absolute{
                                        position: absolute;
                                        bottom: 0px;
                                        right: 0px;
                                    }

Fixed

  • No conserva el espacio reservado.
  • Se posiciona de acuerdo al DOM html y no cambia su posicion al hacer scroll.
Elemento fijo

                                    Codigo:
                                    .element-fixed{
                                        position: fixed;
                                        top: 0px;

Sticky

  • Tiene un espacio reservado con el cual forma parte del flujo normal html, pero se comporta como fixed cuando se cumple una condicion sobre su posicion.
  • Necesita un contenedor con altura, ademas de contar con overflow: visible.
                                
                                    Codigo:
                                    .container-element-sticky{
                                        width: 100%;
                                        height: 10em;
                                        overflow: visible;
                                    }
                                    .element-sticky{
                                        position: sticky;
                                        top: 0px;
                                    }
Nota: Con la propiedad "top" declaramos la condicion que se debe cumplir para que se empiece a comportar como fixed.
