Conectar html y css

Forma 1

Con la etiqueta "link"

Ejemplo ==> link rel="stylesheet" href="/archivo1/styles.css">

Forma 2

Con la etiqueta "style".

Se usa cuando solo se permite cargar un archivo, como por ejemplo una web para correo eletronico.

Ejemplo ==>

style>

body{

background-color: steelblue;

}

/style>

Forma 3

Con @import dentro de la etiqueta "style".

Ejemplo ==>

style>

@import url(css/styles.css);

/style>

Forma 4

Forma incorrecta y anticuada, se usa como propiedad dentro de etiquetas

Ejemplo ==> h1 style="background-color: violet;"> h1>Conectar html y css /h1>
