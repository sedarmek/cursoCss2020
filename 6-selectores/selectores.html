Selectores

  • Inicio

    • Selectores simples

    • Son los selectores que solo tienen una palabra o simbolo.

    Selectores elementales

    Selector universal

    • Es un selector para modificar todos los elementos con un solo caracter "*".
                                        
                                        Codigo:
                                        *{
                                            margin: 0px;
                                        }

    Selector de tipo o etiqueta

    • Es un selector para modificar las etiquetas.
                                        
                                        Codigo:
                                        body{
                                            margin: 0px;
                                        }

    Selectores de atributo

    Selector de id

    • Es un selector para modificar los elementos con el atributo id y el valor especificado.
    • Se coloca un "#" antes del valor del atributo.
    • No se recomienda usar.
                                        
                                        Codigo:
                                        #valor-del-id{
                                            margin: 0px;
                                        }

    Selector de clase

    • Es un selector para modificar los elementos con el atributo class y el valor especificado.
    • Se coloca un "." antes del valor del atributo.
    • Se recomienda usarlo.
                                        
                                        Codigo:
                                        .valor-class{
                                            margin: 0px;
                                        }

    Otros atributos

    Atributo simple
    • Es un selector para modificar los elementos que tengan el atributo especificado.
    • Se coloca un "[" antes y un "]" despues del selector respectivamente.
                                            
                                            Codigo:
                                            [href]{
                                                margin: 0px;
                                            }
    Atributo con valor
    Atributo con valor exacto
    • Busca los elementos que tengan el atributo especificado.
    • Para ello se declara el valor despues del atributo con el signo "=".
                                                
                                                Codigo:
                                                [href=valor]{
                                                    margin: 0px;
                                                }
    Atributo con valor inicial igual
    • Busca los elementos que tengan el atributo especificado y cuyo valor empiece igual.
    • Se usa el signo "^="
                                                
                                                Codigo:
                                                [href^=va]{
                                                    margin: 0px;
                                                }
    Atributo con parte del valor similar
    • Busca los elementos que tengan el atributo especificado y cuyo valor tenga los mismos caracteres, pero que puede tenermos en cualquier posicion.
    • Se usa el signo "*="
                                                
                                                Codigo:
                                                [href*=lo]{
                                                    margin: 0px;
                                                }
    Atributo con valor final igual
    • Busca los elementos que tengan el atributo especificado y cuyo valor termine igual.
    • Se usa el signo "$="
                                                
                                                Codigo:
                                                [href$=or]{
                                                    margin: 0px;
                                                }
    Atributo con o sin guion
    • Busca los elementos que tengan el atributo especificado y cuyo valor inicie igual, incluso si el valor tiene un guion al inicio.
    • Se usa el signo "|="
                                                
                                                [lang|=es-ep]{
                                                    margin: 0px;
                                                }

    Selectores compuestos

    • Son los que manejan mas de un elemento.

    Selectores agrupados

    • Usan comas (",") para trabajar con mas de un elemento.
                                    
                                    Codigo:
                                    h1, h2{
                                        margin: 0px;
                                    }

    Selectores combinadores

    Selector descendente

    • Se usa para seleccionar el hijo de un elemento padre, por buena practica se recomienda no mas de dos.
    • El primer elemento es el padre y el segundo es el hijo.
    • Se representa con el espacio " ".
                                        
                                        Codigo:
                                        ul li{
                                            margin: 0px;
                                        }

    Selector de hermano siguiente

    • Se usa para seleccionar el hermano adyacente, por buena practica se recomienda no mas de dos.
    • El primer elemento es el primer hermano y el segundo es el adyacente.
    • Se representa con el signo "+".
                                        
                                        Codigo:
                                        section+aside{
                                            margin: 0px;
                                        }

    Selector de hermanos siguientes

    • Se usa para seleccionar a todos los hermanos siguientes.
    • El primer elemento es el primer hermano y el segundo son todos los demas.
    • Se representa con el signo "~".
                                        
                                        Codigo:
                                        .primer-li~li{
                                            margin: 0px;
                                        }

    Selector de hijos directos

    • Se usa para seleccionar a solo los hijos directos.
    • El primer elemento es el padre y el segundo son los hijos.
    • Se representa con el signo ">".
                                        
                                        Codigo:
                                        ul>li{
                                            margin: 0px;
                                        }
