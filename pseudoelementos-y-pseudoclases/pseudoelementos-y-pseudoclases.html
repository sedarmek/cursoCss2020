Pseudoelementos y pseudoclases

Inicio

Pseudoelementos

  • Permiten añadir estilos a una parte concreta del documento.
  • Solo se puede usar uno por selector.
  • Debe aparecer después del selector simple.
  • Empieza con "::" o ":".

Primer letra

  • Selecciona la primer letra del elemento
  • Es un elemento de bloque.

Before

  • Selecciona la parte anterior del elemento.
  • La propiedad "content" es obligatoria.
  • Es hijo del elemento al que pertenece.
  • Es un elemento de linea por defecto.

Pseudoclases

  • Es una palabra clave que se añade a los selectores y que especifica un estado especial del elemento seleccionado.
  • Solo se puede usar uno por selector.
  • Debe aparecer después del selector simple.
  • Empieza con ":".

De tipo dinamic

Link no visitado(:link)

Selecciona los enlaces que no hayan sido usados por el usuario.

Link visitado(:visited)

Selecciona los enlaces que ya hayan sido usados por el usuario.

Link activo(:active)

Selecciona los enlaces que esten siendo oprimidos por el usuario.

Sobre(:hover)

Selecciona el elemento que este bajo el puntero del mouse.

Centrado(:focus)

Selecciona el elemento sobre el que se este enfocando.
Link-para-no-visitar Link-para-visitar Link-para-activar
Elemento con hover
                                    
                                        Codigo:
                                        .ej-pseudoclas-link-1:link{
                                            color: blue;
                                        }
                                        .ej-pseudoclas-link-2:visited{
                                            color: gray;
                                        }
                                        .ej-pseudoclas-link-3:active{
                                            color: red;
                                        }
                                        .ej-pseudoclas-hover:hover{
                                            background-color: blueviolet;
                                        }
                                        .ej-pseudoclas-focus:focus{
                                            background-color: pink;
                                        }
