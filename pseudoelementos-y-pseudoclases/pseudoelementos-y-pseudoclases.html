Pseudoelementos y pseudoclases

Inicio

Pseudoelementos

  • Permiten añadir estilos a una parte concreta del documento.
  • Solo se puede usar uno por selector.
  • Debe aparecer después del selector simple.
  • Empieza con "::" o ":".

Primer letra

  • Selecciona la primer letra del elemento
  • Es un elemento de bloque.

Before

  • Selecciona la parte anterior del elemento.
  • La propiedad "content" es obligatoria.
  • Es hijo del elemento al que pertenece.
  • Es un elemento de linea por defecto.

Pseudoclases

  • Es una palabra clave que se añade a los selectores y que especifica un estado especial del elemento seleccionado.
  • Solo se puede usar uno por selector.
  • Debe aparecer después del selector simple.
  • Empieza con ":".

De tipo dinamic

Link no visitado(:link)

Selecciona los enlaces que no hayan sido usados por el usuario.

Link visitado(:visited)

Selecciona los enlaces que ya hayan sido usados por el usuario.

Link activo(:active)

Selecciona los enlaces que esten siendo oprimidos por el usuario.

Sobre(:hover)

Selecciona el elemento que este bajo el puntero del mouse.

Centrado(:focus)

Selecciona el elemento sobre el que se este enfocando.
Link-para-no-visitar Link-para-visitar Link-para-activar
Elemento con hover
                                    
                                        Codigo:
                                        .ej-pseudoclas-link-1:link{
                                            color: blue;
                                        }
                                        .ej-pseudoclas-link-2:visited{
                                            color: gray;
                                        }
                                        .ej-pseudoclas-link-3:active{
                                            color: red;
                                        }
                                        .ej-pseudoclas-hover:hover{
                                            background-color: blueviolet;
                                        }
                                        .ej-pseudoclas-focus:focus{
                                            background-color: pink;
                                        }

De tipo target

Elemento objetivo de ancla(:target)

Selecciona el elemento que es objetivo de marcador o ancla.Y que ya ha sido buscado por el usuario.
Ir al elemento objetivo

Objetivo de ancla: Una vez se haga click en el link que referencia este elemento se cambiara de color a morado.

                                    
                                        Codigo:
                                        #objetivo-ancla:target{
                                            background-color: rebeccapurple;
                                        }

De tipo Language

Idioma o lenguaje(:lang())

Selecciona el elemento que contenga el mismo valor para el atributo lang que especifiquemos.

Text in english.

Texto escrito en español.

                                    
                                        Codigo:
                                        .text-in-lang:lang(en){
                                            color: greenyellow;
                                        }

UI States para formularios

Disponible(:enabled)

Selecciona el elemento que tenga el atributo booleano "enabled".

No disponible(:disabled)

Selecciona el elemento que tenga el atributo booleano "disabled".

Seleccionado(:checked)

Selecciona el elemento que tenga el atributo booleano "checked" o que haya sido seleccionado manualmente por el usuario.

En rango(:in-range)

Selecciona el elemento que este dentro del rango numerico.

Fuera de rango(:out-of-range)

Selecciona el elemento que este fuera del rango numerico.

Opcional(:optional)

Selecciona el elemento que tenga el atributo booleano "optional".

Requerido(:required)

Selecciona el elemento que tenga el atributo booleano "required".

Valido(:valid)

Selecciona el elemento que tenga un valor valido.

Invalido(:invalid)

Selecciona el elemento que tenga un valor que no sea valido.

Solo lectura(:readonly)

Selecciona el elemento que tenga el atributo booleano "readonly".

Lectura y escritura(:read-write)

Selecciona el elemento que tenga el atributo booleano "read-write".

De tipo Structural

Raiz(:root)

Selecciona el elemento raíz del documento(como html, pero con mayor especificidad).Puede ser útil para declarar variables CSS globales.

Vacio(:empty)

Selecciona el elemento que no tenga hijos.

Negacion(:not(selector))

Selecciona el elemento que no coincida con el selector.

Primer hermano(:first-child)

Selecciona el primer elemento entre un grupo de hermanos.

Ultimo hermano(:last-child)

Selecciona el ultimo elemento entre un grupo de hermanos.

Hermano con posicion n(:nth-child(n))

Selecciona el elemento que coincida con la posicion o patron declarado en "n" entre un grupo de hermanos, contando desde el principio.

Hermano con posicion n invertida(:nth-last-child(n))

Selecciona el elemento que coincida con la posicion o patron declarado en "n" entre un grupo de hermanos, contando desde el final.

Hijo unico(:only-child)

Selecciona el elemento que no tenga hermanos.

Primer hermano en su tipo(:first-of-type)

Selecciona el primer elemento de su tipo entre un grupo de elementos hermanos.

Ultimo hermano en su tipo(:last-of-type)

Selecciona el ultimo elemento de su tipo entre un grupo de elementos hermanos.

Hermano con posicion n del mismo tipo(:nth-of-type(n))

Selecciona el elemento que coincida con la posicion o patron declarado en "n" entre un grupo de hermanos del mismo tipo, contando desde el principio.

Hermano con posicion n invertida del mismo tipo(:nth-last-child(n))

Selecciona el elemento que coincida con la posicion o patron declarado en "n" entre un grupo de hermanos del mismo tipo, contando desde el final.

Hijo unico en su tipo(:only-of-type)

Selecciona el elemento que no tenga hermanos de su tipo.
Valores posibles de n:
  • odd(numeros impares)
  • even(numeros pares)
  • numeros enteros
  • ecuaciones(donde la letra n es un multiplicador)
  • Css
  • Curso Html 2020
  • JS
